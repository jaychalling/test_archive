# 테스트 기획서: IQ Test (미국 시장용)

## 기본 정보

- **test-id**: `iq-test`
- **카테고리**: Personality
- **예상 소요시간**: 20-30분
- **질문 수**: 50문항
- **타겟 시장**: 미국 (US English, $)

## 테스트 개요

미국 디지털 평가 시장 타겟 50문항 IQ 테스트. 웩슬러(Wechsler) 모델과 멘사(Mensa) 스타일 레이븐 매트릭스를 3:3:4 비율로 결합.

**핵심 목표:**
- 학문적 신뢰성 확보
- 무료 테스트 신뢰성 부족 문제 해결
- 유료 리포트 전환율 극대화

## 테스트 유형

- [x] **B. 점수 합산** - 총점 기반 IQ 산출

## 점수 산출 알고리즘

### IQ 환산 공식
```
IQ = 100 + 15 × (Raw Score - μ) / σ
```
- μ (예상 평균): 28점
- σ (표준편차): 10점
- 신뢰구간: 95% 수준에서 ±3~5점
